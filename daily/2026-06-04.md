## [How we reduced core unit boot time from hours to minutes](https://blog.cloudflare.com/optimizing-core-unit-boot-time/) 🤓

Cloudflare's firmware upgrade process requires multiple sequential reboots across servers in 285+ cities — and a debugging investigation revealed that each reboot silently wasted up to 20 minutes on failed UEFI network boot attempts before handing control back to the OS. UEFI was attempting IPv4 HTTPS boot, timing out after ~5 minutes, then falling back to IPv4 iPXE, timing out again — four failed attempts stacking to ~20 minutes per reboot cycle, which compounded across sequential per-component firmware upgrades into nearly four hours of idle waiting per server. The fix required diving into UEFI data structures, collaborating with OEM vendors to unlock programmatic boot order control, and building iPXE-based automation that eliminates the timeout cascade entirely. #devops #engineering #architecture #cloud

## [Under the River](https://shopify.engineering/under-the-river) 👨‍💼

Shopify's Aquifer is an internal platform for running production AI agents, composed of six integrated layers: session, harness, sandbox, gateway, durable event log, credentials proxy, and observability pipeline. River — the Slack-native agent built on Aquifer — ran 59,918 sessions across 5,170 Slack channels touching 7,000+ Shopify employees in a single month, and now co-authors one in every eight merged pull requests company-wide. The decisive architectural choice is separating the agent's "brain" (model conversation state, persisted durably in the event log) from its "hands" (sandboxed code execution environment), enabling long-running agent sessions that survive restarts and remain fully inspectable after the fact. #ai #engineering #architecture #softwareengineering

## [Top Agentic Frameworks for Building Applications 2026](https://blog.jetbrains.com/pycharm/2026/06/top-agentic-frameworks-for-building-applications-2026/) 🍼

Agentic frameworks have matured from experimental scaffolding into a required architectural layer — with LangGraph establishing itself as the production standard by modeling agents as directed graphs where typed state persists across nodes, conditional edges route execution dynamically, and checkpoints enable human-in-the-loop interrupts at any step. CrewAI takes a role-based approach, assigning each agent a role, goal, and backstory within a crew that executes via sequential, hierarchical (manager delegates to workers), or consensual (vote-based) process modes. The survey maps LangGraph, CrewAI, AutoGen, and newer entrants against control requirements and team complexity, giving Python teams a practical selection guide for choosing orchestration primitives in 2026. #ai #development #engineering #architecture
