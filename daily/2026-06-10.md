## [Project Glasswing: What Mythos Showed Us](https://blog.cloudflare.com/cyber-frontier-models/) 🤓

Running Anthropic's Mythos Preview model against 50+ internal repositories, Cloudflare found 2,000 bugs — 400 high or critical severity — at a false-positive rate beating human testers, with the model reasoning across multiple code primitives to construct working exploit chains, write proof-of-concept code, compile it, and run it autonomously. Across 50 partner organizations, Project Glasswing surfaced over 10,000 high- or critical-severity vulnerabilities, demonstrating that AI has effectively solved the discovery problem at scale. The new bottleneck is verification, disclosure, and patch velocity: no existing security workflow can absorb the vulnerability volume AI now generates. #engineering #ai #softwareengineering

## [Defend Against Frontier Cyber Models: Cloudflare's Architecture as Customer Zero](https://blog.cloudflare.com/frontier-model-defense/) 👨‍💼

Cloudflare's defense-in-depth against AI-powered attackers replaces signature-based WAF with ML scoring (1–99 based on attack shape similarity), enabling rulesets to reach the entire global network within 30 seconds of a proof-of-concept dropping — before most vendors have published a CVE. API Shield enforces a positive security model that restricts traffic to described API shapes, so generating thousands of novel attack variants fails to bypass it. The core architectural lesson, validated by running this on Cloudflare's own infrastructure as "customer zero": architecture around a vulnerability limits blast radius more reliably than patch speed does. #architecture #engineering #ai #softwareengineering

## [Security Validation for Third-Party Coding Agents](https://github.blog/changelog/2026-06-09-security-validation-for-third-party-coding-agents/) 👨‍💼

GitHub now automatically runs CodeQL code scanning, dependency advisory checks (GitHub Advisory Database, CVSS High/Critical), and secret scanning against every pull request created by a third-party coding agent — including Claude and OpenAI Codex — with the agent attempting auto-remediation before the PR is finalized. No GitHub Advanced Security license is required, and since the October 2025 rollout for Copilot cloud agent the feature has proactively prevented hundreds of potential security leaks. This closes the supervision gap that made agentic CI/CD risky: code generated without human review now receives the same security gates as human-authored code. #engineering #ai #softwareengineering #devops
